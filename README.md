Features
--------------------
* Cross-browser compatible
* Two-column design
* Animated Image Slider
* Fixed-layout
* Threaded comments
* Translation-ready
* Pagination
* Customizable header image
* Customizable background
* Main colors: white, blue, gray

Translation
